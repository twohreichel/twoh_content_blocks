============
Installation
============

1. Install via Composer:

.. code-block:: bash

   composer require twoh/twoh_content_blocks

2. Include the TypoScript in the **ROOT Template**:

   *Site Configuration → TypoScript → Includes → TWOH Content Blocks*

3. Install dependency:

.. code-block:: bash

   composer require contentblocks/content-blocks

That's all. The new content elements will appear in the "New Content Element"
wizard under the group *Common*.
